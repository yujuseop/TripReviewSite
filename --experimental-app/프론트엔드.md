## 커스텀 훅 (Hooks)

| 파일 | 역할 |
|------|------|
| `hooks/useAuth.ts` | 로그인 / 회원가입 / 로그아웃 |
| `hooks/useDashboard.ts` | 대시보드 전체 상태 (여행 목록, 모달, 필터) |
| `hooks/useTravelForm.ts` | 여행 추가 폼 상태 (useReducer 기반) |
| `hooks/useDelete.ts` | 여행 / 리뷰 삭제 |
| `hooks/useReviewEdit.ts` | 리뷰 수정 |
| `hooks/useModal.ts` | 모달 열기/닫기 상태 |
| `hooks/useModalState.ts` | 모달 상태 상세 관리 |

## 주요 기능 흐름

### 여행 추가 플로우

1. 캘린더에서 날짜 범위 선택 → `travelModal` 열기
2. 여행 제목 / 위치 / 목적지 / 리뷰 / 이미지 / 평점 입력
3. `useTravelForm` (useReducer) 으로 폼 상태 관리
4. 제출 → DB 저장 후 목록 갱신

### 리뷰 수정 / 삭제 플로우

1. `TravelList`에서 항목 클릭 → `ReviewDetailModal` 표시
2. 수정 버튼 → `ReviewEditModal` (내용 + 평점 변경)
3. 삭제 버튼 → `DeleteConfirmModal` (확인 후 삭제)

### 인증 플로우

1. 폼 입력 → Zod 스키마로 실시간 검증
2. 제출 → `useAuth` 훅 → Supabase Auth API 호출
3. JWT 쿠키 저장 → 미들웨어에서 세션 갱신

## 스타일링 규칙

- Tailwind CSS v4 사용 (PostCSS 플러그인 방식)
- `globals.css` 에서 기본 스타일 정의
- 반응형: `md:` 브레이크포인트 기준
- 절대 경로: `@/` prefix 사용

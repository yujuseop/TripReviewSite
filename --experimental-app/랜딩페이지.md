# 랜딩 페이지

## 파일 구조

```
app/page.tsx
  ├── components/landing/HeroSlider.tsx
  ├── components/landing/FeatureSection.tsx
  └── components/landing/CtaSection.tsx
```

## 섹션 구성

### 1. HeroSlider (`components/landing/HeroSlider.tsx`)

- `"use client"` — useState / useEffect 사용
- **슬라이드 이미지**: `public/image/` 폴더의 9장 사진 자동 순환 (4초 간격)
- **전환 효과**: CSS `transition-opacity duration-1000` 크로스페이드
- **오버레이**: `bg-black/50` 반투명 어두운 레이어
- **텍스트**: TripView 타이틀 + 소개 문구
- **버튼**: `AuthButtonsSupabase` — 로그인 상태에 따라 다른 버튼 표시
- **하단 인디케이터**: 현재 슬라이드 점(dot) + 클릭으로 이동 가능
- **위치 레이블**: 현재 슬라이드 지역명 우측 하단 표시

**슬라이드 이미지 목록:**

| 파일 | 레이블 |
|------|--------|
| `강원도.jpeg` | 강원도 정동진 |
| `경기도 가평.jpeg` | 경기도 가평 |
| `대구.jpeg` | 대구 팔공산 |
| `롯데월드&타워.jpeg` | 서울 롯데월드 & 타워 |
| `영월.jpeg` | 영월 청령포 |
| `전라도 전주.jpeg` | 전라도 전주 |
| `진해.jpeg` | 진해 군항제 |
| `충청도.jpeg` | 충청도 단양 |
| `해운대.jpeg` | 부산 해운대 |

### 2. FeatureSection (`components/landing/FeatureSection.tsx`)

- 서버 컴포넌트 (정적 콘텐츠)
- 흰색 배경 (`bg-white`)
- 4개 기능 카드 그리드 (1열 → md: 2열 → lg: 4열)

| 아이콘 | 제목 | 내용 |
|--------|------|------|
| ✈️ | 여행 기록 | 날짜 / 목적지 체계적 관리 |
| ⭐ | 리뷰 & 평점 | 후기와 별점, 수정/삭제 가능 |
| 🖼️ | 사진 갤러리 | 여행 사진 업로드 및 모아보기 |
| 📅 | 캘린더 뷰 | 여행 날짜 한눈에 확인 |

---

### 3. CtaSection (`components/landing/CtaSection.tsx`)

- `"use client"` — `useAuth` 훅으로 로그인 상태 확인
- 어두운 배경 (`bg-gray-900`)
- **로그인 상태**: "내 여행 보러가기" → `/dashboard`
- **비로그인 상태**: "시작하기" → `/login`
- 로딩 중 버튼 숨김 (잘못된 페이지 이동 방지)

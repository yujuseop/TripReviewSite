## 아키텍처 패턴

- **App Router** — Next.js 13+ 파일 기반 라우팅
- **Server Components** — 서버에서 렌더링 (초기 데이터 패칭)
- **Client Components** — `"use client"` 지시어 (인터랙션 처리)
- **dynamic import** — SSR 비활성화 필요한 컴포넌트 (leaflet, 카카오맵)
- **Middleware** — 모든 요청에서 Supabase 인증 상태 갱신
- **절대 경로 alias** — `@/*` → 프로젝트 루트

## 환경 변수

```env
NEXT_PUBLIC_SUPABASE_URL          # Supabase 프로젝트 URL
NEXT_PUBLIC_SUPABASE_ANON_KEY     # Supabase anon 키
SUPABASE_SERVICE_ROLE_KEY         # Supabase 서비스 롤 키 (서버 전용)
NEXT_PUBLIC_ADMIN_CODE            # 관리자 회원가입 코드
NEXT_PUBLIC_KAKAO_MAP_KEY         # 카카오맵 JavaScript API 키
```

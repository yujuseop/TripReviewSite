# 백엔드

## 개요

별도의 백엔드 서버 없이 **Supabase(BaaS)**를 백엔드로 사용합니다.  
Next.js 서버 컴포넌트 / 미들웨어에서 서버 클라이언트, 클라이언트 컴포넌트에서 브라우저 클라이언트를 사용합니다.

## Supabase 클라이언트

| 파일 | 용도 |
|------|------|
| `lib/supabaseClient.ts` | 브라우저(클라이언트 컴포넌트)용 Supabase 인스턴스 |
| `lib/supabaseServer.ts` | 서버(서버 컴포넌트, 미들웨어)용 Supabase 인스턴스 |

## 미들웨어 (middleware.ts)

- 모든 요청(정적 파일 제외)에서 실행
- Supabase 세션/쿠키 갱신
- 인증 상태 확인 후 보호된 라우트 접근 제어

## 데이터베이스 스키마

### profiles 테이블

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| user_id | uuid | Supabase Auth 사용자 ID (FK) |
| nickname | string | 닉네임 |
| role | string | 역할 (user / admin) |
| profile_Image | string | 프로필 이미지 URL |
| bio | string | 자기소개 |
| created_at | timestamp | 생성일시 |
| updated_at | timestamp | 수정일시 |

### trip 테이블

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| user_id | uuid | Supabase Auth 사용자 ID (FK) |
| title | string | 여행 제목 |
| start_date | string | 시작일 (YYYY-MM-DD) |
| end_date | string | 종료일 (YYYY-MM-DD) |
| description | string | 여행 설명 |
| is_public | boolean | 공개 여부 |
| created_at | timestamp | 생성일시 |

### destinations 테이블

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| trip_id | uuid | trip FK |
| name | string | 목적지 이름 |
| description | string | 목적지 설명 |
| day | number | 여행 일차 (1, 2, 3...) |
| order_num | number | 같은 날 내 순서 |
| **category** | text | 카테고리 (맛집/관광지/카페/숙소/액티비티/쇼핑/기타) |
| **lat** | float8 | 위도 (카카오맵 선택) |
| **lng** | float8 | 경도 (카카오맵 선택) |
| **cost** | numeric | 해당 목적지 비용 (원) |
| created_at | timestamp | 생성일시 |

### reviews 테이블

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| trip_id | uuid | trip FK |
| user_id | uuid | Supabase Auth 사용자 ID |
| content | string | 리뷰 내용 |
| rating | number | 평점 (1~5) |
| images | string[] | Supabase Storage 이미지 URL 배열 |
| **mood_tags** | text[] | 무드 태그 (행복함/여유로움 등) |
| **one_line_summary** | text | 한 줄 평 |
| created_at | timestamp | 생성일시 |

> **굵게** 표시된 컬럼이 고도화 작업에서 추가된 항목입니다.

### 스키마 추가 SQL

```sql
ALTER TABLE destinations
  ADD COLUMN IF NOT EXISTS category text,
  ADD COLUMN IF NOT EXISTS lat float8,
  ADD COLUMN IF NOT EXISTS lng float8,
  ADD COLUMN IF NOT EXISTS cost numeric;

ALTER TABLE reviews
  ADD COLUMN IF NOT EXISTS mood_tags text[],
  ADD COLUMN IF NOT EXISTS one_line_summary text;
```

## 인증 (Supabase Auth)

- 이메일 + 비밀번호 방식
- 회원가입 시 사용자 메타데이터에 `nickname`, `role` 저장
- JWT 토큰 쿠키 방식으로 세션 유지
- 관리자 회원가입: `NEXT_PUBLIC_ADMIN_CODE` 환경변수 값 일치 시 `role: admin` 부여

## 파일 스토리지 (Supabase Storage)

- 버킷: `trip-images`
- 경로 구조: `{userId}/{tripId}/{timestamp}-{index}.{ext}`
- 이미지 URL: `{SUPABASE_URL}/storage/v1/object/public/trip-images/{path}`
- `next.config.ts`에서 `**.supabase.co` 도메인 Next.js Image 최적화 허용

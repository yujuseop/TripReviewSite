# 여행 폼

## 파일 구조

```
components/dashboard/
  ├── travelModal.tsx                 ← 모달 껍데기 + DB 저장 로직
  └── travelForm/
      ├── LocationInput.tsx           ← 여행 제목/지역 입력
      ├── DestinationSection.tsx      ← 목적지 추가 (카테고리, 비용, 지도)
      ├── ReviewSection.tsx           ← 리뷰 작성 (한 줄 평, 무드 태그)
      ├── RatingSection.tsx           ← 별점 선택
      ├── ImageSection.tsx            ← 이미지 업로드
      ├── KakaoMapPicker.tsx          ← 카카오맵 위치 선택기
      └── PublicToggle.tsx            ← 공개/비공개 (현재 주석 처리)
```

## 상태 관리 — useTravelForm / TravelFormState

| `destDay` | number | 입력 중인 여행 일차 |
| `destCost` | number | 입력 중인 비용 |
| `destCategory` | string | 입력 중인 카테고리 |

| `destLng` | number \| null | 선택된 경도 |

## ReviewSection

**입력 항목:**
1. **한 줄 평** (`one_line_summary`) — text input
2. **상세 리뷰** (`reviewContent`) — textarea (3줄)
3. **무드 태그** (`moodTags`) — 칩 스타일 멀티 선택 버튼

**무드 태그 목록:** 행복함 / 여유로움 / 설렘 / 피곤함 / 아쉬움 / 활기참 / 낭만적 / 바쁨

선택된 태그: `bg-purple-500 text-white` / 미선택: `border-gray-300`

## DestinationSection

**입력 항목:**
1. 목적지 이름 + 일차 선택 (1~7일차)
2. 카테고리 드롭다운 + 비용 입력 (원)
3. 세부사항 (선택사항)
4. **🗺 지도에서 위치 선택** — KakaoMapPicker 토글

**카테고리 목록:** 관광지 / 맛집 / 카페 / 숙소 / 액티비티 / 쇼핑 / 기타

**지도 위치 선택 흐름:**
```
[🗺 지도에서 위치 선택] 버튼 클릭
  → KakaoMapPicker 펼쳐짐 (dynamic import, ssr: false)
  → 장소 검색 or 지도 직접 클릭
  → 📍 [장소명] 선택됨 배지 표시
  → [+ 목적지 추가] 클릭 → lat/lng 포함 저장 + 지도 닫힘
```

추가된 목적지에 좌표가 있으면 📍 아이콘 표시.

## travelModal — DB 저장 흐름

```
1. trip 테이블 INSERT (제목, 날짜, 공개여부)
2. destinations INSERT (이름, 일차, 카테고리, 비용, lat, lng)
3. 이미지 Supabase Storage 업로드
4. reviews INSERT (내용, 별점, 이미지URLs, mood_tags, one_line_summary)
5. onTravelAdded 콜백 → 대시보드 목록 즉시 갱신
```

리뷰는 `reviewContent` 또는 `oneLineSummary` 중 하나라도 있으면 저장.
